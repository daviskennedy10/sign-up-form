## 📋 ZEN Registration Form
A clean, split-screen user sign-up interface featuring a cinematic left-side branding panel and a dual-column registration form on the right.
## 🛠️ Tech Stack & Features

* HTML5 & CSS Flexbox: Responsive, fluid dual-panel layout.
* Native Validation: Dynamic border updates (green for valid, red for invalid) using input:user-valid and input:user-invalid.
* Interactive UI: Smooth blue border glows on active <input> fields.
* Custom Typography: Integrated geometric "Norse" headline fonts via @font-face.

## 📂 File Structure

* index.html — Form structure with input constraints (minlength, required).
* styles.css — Flexbox layouts, focus states, and typography.
* odin-lined.png — Core vector branding logo.
* Norse.otf / Norsebold.otf — Custom asset font files.

## 🔧 How to Run

   1. Keep all files in the same folder so image and font paths link correctly.
   2. Double-click index.html to open and test the form in any web browser.
